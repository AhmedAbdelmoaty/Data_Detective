## Packages
zustand | State management for the game engine
recharts | Charts for the data analysis view
framer-motion | Animations for transitions and UI effects
lucide-react | Icons (already in base, but ensuring availability)
date-fns | Date formatting for reports

## Notes
The application must be strictly RTL (Right-to-Left).
All content is in Arabic.
Game state is primarily client-side for this prototype, using Zustand.
Mock data is defined in `client/src/content/cases/case001.ts`.
